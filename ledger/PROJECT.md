# PlantCare

## Overview
A comprehensive iOS app for managing a plant collection with intelligent care reminders, AI-powered plant identification, and organised space management (rooms and outdoor zones). Built primarily with Claude Code as a personal tool.

## Tech Stack
- **Platform:** iOS (Swift, SwiftUI)
- **IDE:** Xcode
- **AI:** OpenAI API (plant identification, care recommendations, health checks)
- **Storage:** Local + iCloud backup
- **Min iOS:** 16.0+

## Architecture
- Xcode project (`PlantCare.xcodeproj`)
- SwiftUI views, local data persistence
- OpenAI integration for AI features (identification, care advice, health analysis)
- iCloud Documents for backup/sync
